Dynamic Scorecard Tool
======================

This is a web-based scorecard tool for evaluating and displaying performance metrics. It allows users to upload data, calculate scores, and visualize results.

Instructions:
1. Install Python 3.x and pip.
2. Navigate to the backend folder and install dependencies: 
3. Run the Flask application:
4. Open your browser and navigate to http://127.0.0.1:5000/.
5. Upload an Excel/CSV file to get started.

Features:
- Upload Excel/CSV files.
- Dynamic weight calculation.
- Multiple visualizations (bar chart, pie chart, radar chart).
- Export scorecard as PDF, Excel, or CSV.
- Send scorecard via email.
